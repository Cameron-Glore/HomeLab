# HomeLab

## About Me
My name is Cameron, and this repository highlights hands-on infrastructure, monitoring, networking, and security labs I built while transitioning into IT and cybersecurity.

I am currently working toward an associate degree in networking technologies and already hold two associate degrees in science. Most of my growth has come through hands-on lab work, self-study, and real-world troubleshooting experience.

In my current role, I support day-to-day technology operations for a small business, including troubleshooting computers, store systems, printers, vendor-related issues, and general technical problems. I built this repository to document practical projects, strengthen my skills, and show my progression through real hands-on work.

## Featured Projects

### [CrowdSec VPS Defense Lab](./CrowdSec-VPS-Defense-Lab)
Deployed CrowdSec on a live VPS to monitor logs, detect SSH brute-force activity, and send security alerts to Discord.

### [Pi5 Private Security Operations Node](./Pi5-Private-Security-Operations-Node)
Built a private Raspberry Pi 5 operations node using Docker, Portainer, Dozzle, Tailscale, and CrowdSec for secure remote administration and monitoring.

### [Pi5 Tailscale Admin Hardening Lab](./Pi5-Tailscale-Admin-Hardening-Lab)
Hardened self-hosted admin services by restricting access to a private Tailscale network instead of exposing them publicly.

### [Service Monitoring and Mobile Alerting Lab](./service-monitoring-mobile-alerting-lab)
Deployed uptime monitoring and mobile alerting for VPS services to improve visibility and response to outages.
